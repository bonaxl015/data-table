# DataTable

DataTable component with different props.

#### Example

<Canvas of={DataTable.BasicUsage} />

## Usage

```ts
import { DataTable } from "sld-ui";

const Example = () => {
  return (
      <DataTable
        rowKey="id"
        columns={[]}
        dataSource={[]}
        searchFields={[]}
      />
  );
};

export default Example;
```

#### Arguments

- **rowKey** _`() => void`_ - A string that represents the text content of the button.
- **columns** - A boolean indicating whether the button should have a primary styling or not. Typically, a primary button stands out as the main action in a user interface.
- **dataSource** - A boolean indicating whether the button should be disabled or not. When disabled, the button cannot be clicked or interacted with.
- **searchFields** - A string with one of three possible values: "small," "medium," or "large." It defines the size or dimensions of the button.
- **onSearch** - A function that is called when the button is clicked. It receives a MouseEventHandler for handling the click event on the button element.
